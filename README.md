# cadastro-clientes-python
Projeto em Python - sistema de cadastro de clientes

Este projeto consiste em um sistema de cadastro de clientes desenvolvido em Python, executado via terminal. O objetivo é permitir o gerenciamento básico de dados, possibilitando o cadastro, a listagem e a busca de clientes de forma prática e organizada.

O sistema oferece as seguintes funcionalidades:

- Cadastro de novos clientes (nome, idade e e-mail)
- Listagem de todos os clientes cadastrados
- Busca de cliente pelo nome
- Menu interativo no terminal
- Validação básica de opções do usuário.
